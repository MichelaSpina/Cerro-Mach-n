Questa repository contiene:
- analysis.Rmd : R Markdown completo per ritaglio raster, calcolo indici e analisi dati vegetali.
- render.R : script semplice per eseguire `rmarkdown::render("analysis.Rmd")`.

Requisiti
- R >= 4.0
- Pacchetti: terra, viridis, readr, dplyr, ggplot2, rmarkdown, knitr (vedi header Rmd per installazione)
